# Design Tokens: FK STUDIO - Orange Accent Edition

## 1. Typography
The variant utilizes the following font exclusively for a clean, brutalist aesthetic:
*   **Headline/Body/Label Font:** Manrope
*   **Icons:** Material Symbols Outlined

## 2. Spacing & Shapes
*   **Border Radius:** Uses specific scale from `rounded-lg` (2rem) up to `rounded-full` (9999px).

## 3. Color Palette (Dark Mode)
The following hex codes define the vibrant, high-contrast palette:

### Background & Surfaces
*   `background`: `#131313`
*   `surface`: `#131313`
*   `surface_container_lowest`: `#0e0e0e`
*   `surface_container_low`: `#1b1b1b`
*   `surface_container`: `#1f1f1f`
*   `surface_container_high`: `#2a2a2a`
*   `surface_container_highest`: `#353535`

### Primary & Accent Elements
*   **`accent` (New Core Identifier):** `#F85F1E` (Vibrant Orange)
*   `primary`: `#ffffff`
*   `on_primary`: `#1a1c1c`
*   `secondary`: `#c6c6c6`
*   `outline`: `#919191`
*   `outline_variant`: `#474747`

## 4. Textures
*   **`hero-grid-bg`**: A linear-gradient grid pattern creating architectural blueprints vibes on dark backgrounds.
